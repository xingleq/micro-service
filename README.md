# xing-micro-service 个人微服务
## 模块说明

```
xing-micro-service -------------- 父项目，公共依赖
│  │─xing-commons --------------- 通用工具
│  ├─xing-config ---------------- 配置中心
│  ├─xing-gateway --------------- api网关
│  ├─xing-job ------------------- 分布式任务调度
│  ├─xing-monitor --------------- 监控
│  ├─xing-register -------------- 注册中心
│  ├─xing-security -------------- 认证中心
│  ├─xing-service --------------- 业务模块
```
